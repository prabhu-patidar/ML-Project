# ML-Project
Machine Learning project with intelligent validation and confidence-based predictions across numeric, text, image, audio, and video data using Streamlit.
## How to Run

python -m streamlit run app.py

## Modules

- Heart Disease Prediction (Numeric Data)
- Movie Sentiment Analysis (Text Data)
- Flower Image Classification
- Voice Gender Recognition
- Workout Detection (Video)

## Features

- Input validation
- Confidence-based predictions
- Error handling for invalid inputs
- Streamlit GUI interface
